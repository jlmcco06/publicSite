    PEEPAL

    PeePal is a responsive single-page web application, built to help folks find the best public restrooms in their time of need!

    Users can search for restrooms by rating, distance, or can simply draw a search area on the map. Clicking on the toilet-paper icon opens a location profile where photos, wikipedia summarys, hours of operation, and other user reviews of each location can be found. Once a location has been chosen, the user simply clicks the directions button, and Google provides walking directions to the chosen location. Then a users can add their own rating to the location to help others decide later!

    PeePal works with the Google Maps, Flickr and Wikipedia APIs, and was built with KnockoutJS.
